# Step2Win — Design System

Simple, clean, intuitive. Mobile-first. Bilingual (RU / KK).

## Colors
- Primary: `#2563EB` (blue) — CTAs, active states, links
- Primary dark: `#1E40AF`
- Accent: `#F59E0B` (amber) — XP, badges, highlights
- Success: `#16A34A` — completed, passed
- Danger: `#DC2626` — errors, failed
- Background: `#F8FAFC` (light gray-blue)
- Surface (cards): `#FFFFFF`
- Border: `#E2E8F0`
- Text primary: `#0F172A`
- Text secondary: `#64748B`

## Typography
- Font: Poppins (headings, weight 600-700), Inter (body text, weight 400-500)
- H1: 32-40px / bold
- H2: 24px / semibold
- Body: 16px / regular
- Small: 14px / regular, secondary color

## Layout
- Max content width: 1200px, centered
- Card radius: 12px, subtle shadow (`shadow-sm`)
- Spacing scale: 4/8/12/16/24/32/48px
- Mobile-first grid, stacks to columns on `md:` breakpoint

## Components
- Buttons: solid primary (blue, white text, rounded-lg), outline secondary
- Nav: top bar with logo, language switch (RU/KK), avatar/menu
- Cards: white surface, border, rounded-12, used for courses/tests/projects/events
- Progress bar: thin rounded bar, primary fill
- Badges: pill shape, colored by status (idea=gray, hypothesis=blue, mvp=amber, demoday=green)
- Tabs: underline style for switching sections (Courses/Tests/Projects/Events)

## Tone
Friendly, direct, encouraging — for school-age users (5-11 grade) and mentors/admins. No jargon-heavy corporate copy. Short labels, clear icons (lucide-react).
